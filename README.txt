Simple Calculator Block for Moodle
===================================

This plugin provides a basic calculator block within any Moodle course.
It includes the essential arithmetic operations and supports decimal input.

Features:
---------
- Addition, subtraction, multiplication, division
- Decimal (comma-based) input
- Clear function
- Fully functional visual layout as tested
- Works without JavaScript libraries

Installation:
-------------
1. Upload the ZIP via Site administration > Plugins > Install plugins
2. Clear caches after installation.
3. Add the block to any course page.
